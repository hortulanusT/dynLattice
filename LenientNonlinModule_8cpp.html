Detailed Description

Author
F.P. van der Meer, T. Gärtner

This module implements a nonlinear solver that can accept non-converged solutions when a maximum iteration limit is reached. It extends the standard nonlinear solver with relaxed convergence criteria for difficult problems.

Original implementation: F.P. van der Meer, April 2014 Modified: T. Gärtner, January 2025

  • Accept solution if not converged directly
  • Check maximum iteration limit before accepting
