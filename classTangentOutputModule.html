Detailed Description

Computes effective elastic properties of heterogeneous materials using periodic boundary conditions. Supports two computation modes:

  • Finite differences: Applies strain perturbations and measures stress response
  • Matrix condensation: Direct extraction from tangent stiffness matrix *(NOT IMPLEMENTED)*

The module performs strain-stress analysis on boundary nodes and calculates homogenized material properties for multi-scale modeling applications.
