Rod-based Lattice Simulations

This repository contains JEM/JIVE code and corresponding scripts to execute a FEA based on nonlinear Beam elements implemented after Simo, Vu-Quoc (1986) as well as some documentation.

Requirements

You can use the code in this repository with the provided Apptainer container. To run the code, you need to have Apptainer installed on your system. The code is built using the Makefile in the root directory. Using the provided container, you can build also build the documentation using <apptainer exec jive.sif> make doc.
